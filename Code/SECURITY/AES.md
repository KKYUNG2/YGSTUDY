# AES
- 표준이었던 DES를 대신하여 새로운 표준이 된 대칭 알고리즘이다.
- 전 세계의 기업과 암호학자가 AES의 후보로서 다수의 대칭 암호 알고리즘을 제안했지만!!
- 그 중에 Rijndel이라는 대칭 암호 알고리즘이 2000년에 AES로 선정

# AES가 DES를 대신한 이유
1. DES에 비해서 키 사이즈가 자유롭다.
2. 즉, 가변 길이의 블록과 가변 길이의 키 사용이 가능하다.(128bit, 192bit, 256bit)
